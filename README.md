## Usage

- git clone https://github.com/Timson020/Game-Builder.git

- npm i babel-cli -g (已经安装可以忽略)

- npm i && npm run init

- npm run release && npm run dev

- 打开目录下面dist目录下的html文件即可

## Cmd (git-bash || linux-cmd)

- npm run dev （启动开发模式）

- npm run release （打生产环境源码）
